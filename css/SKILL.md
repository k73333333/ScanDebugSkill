# Sonar CSS 代码质量与安全修复技能

此技能用于指导 AI 助手协助用户修复 SonarQube 及奇安信安全扫描发现的 CSS/SASS/LESS 样式代码问题。

## 核心原则
1.  **直接高效**：直接给出修复后的完整代码片段，**始终保持中文输出**。
2.  **规范统一**：严格遵守项目的代码风格与注释规范。
3.  **兼容性优先**：考虑不同浏览器的兼容性回退方案。
