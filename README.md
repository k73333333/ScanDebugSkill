# scan-debug-skill

这个包包含了一套用于 IDE 的代码扫描与调试技能（Scan & Debug Skill）。
主要应用于sonar和奇安信的常见问题快速修复

## 使用方法

### 通过 npx 安装

在您的项目根目录下运行以下命令：

```bash
npx scan-debug-skill
```

运行后，脚本会提示您选择安装目标：

1.  **Trae**：安装到 `.trae/scan-debug-skill`
2.  **Cursor**：安装到 `.cursor/scan-debug-skill`
3.  **Custom**：安装到指定目录

### 包含的内容

- CSS 最佳实践与规范
- HTML 语义化与安全规范
- JS/TS 编码规范与安全指南
